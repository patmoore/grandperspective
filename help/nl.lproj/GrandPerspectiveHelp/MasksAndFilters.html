Filters en maskers

U kunt bestanden in een overzicht venster verbergen door middel van een filter of masker:

  • Een filter kan gebruikt worden om bestanden uit te sluiten van een venster. Bestanden die niet aan het filter voldoen worden weggelaten. Wanneer een filter wordt toegepast, wordt altijd een nieuw venster gecre‘rd. U kunt een filter toepassen op een bestaand overzichts venster via de Filter opdracht in het "Venster" menu. U kunt een filter gebruiken tijdens het inlezen van een nieuwe map door middel van Map inlezen met filter in het "Archief" menu.
  • Een masker is ook een filter, maar wordt iets anders gebruikt. Een masker wordt op een bestaand overzicht venster toegepast. Een masker verbergt bestanden, maar ze nemen wel nog steeds ruimte in binnen het venster. U kunt een masker toepassen via het Beeld paneel van het venster.
